# Grocery Shop E-Commerce (CodeIgniter 3)

## Overview
A full-featured grocery e-commerce web application developed using CodeIgniter 3.  
The system supports product management, categories, cart, checkout, and admin management.

This project is built to demonstrate real-world e-commerce architecture using MVC principles.

## Features
### User
- User registration & login
- Product browsing by category
- Add to cart & cart management
- Order placement & checkout
- Order history

### Admin
- Secure admin login
- Category & product management
- Order management
- User management
- Banner & offer management

## Tech Stack
- CodeIgniter 3
- PHP 7.x
- MySQL
- HTML5, CSS3
- Bootstrap
- jQuery
